# Archives

This folder stores files moved out of active runtime/documentation paths during repository cleanup.

## Structure

- `Docs/` — historical or supplementary documents no longer part of the active docs set.
- `Codes/` — legacy/obsolete code and non-runtime artifacts.
- `Tests/` — archived test files (currently none moved).

## Notes

- Active documentation remains in `docs/`.
- Active runtime code remains in `hacker_ai/`, `services/`, and `infrastructure/`.
- Archived files are preserved for traceability and can be restored if needed.
